## 🖥️ <b>How to install ?
```
npm install crickrandom
```
   
<br>
 
# 🎉 Functions! 🎉
```js
var crickplayer = require('crickrandom');

console.log(cricketplayer(1));
\\It will reply a cricket player

console.log(crickplayer(5));
\\It will reply 5 cricket players
```
